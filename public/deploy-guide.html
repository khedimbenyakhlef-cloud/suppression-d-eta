ÉTAPE 1 — GITHUB

1
Initialiser le dépôt Git
cd suppression-deta-game
git init
git add .
git commit -m "Suppression Déta v1.0.0 — Jeu complet"
2
Créer le repo sur GitHub
Sur github.com/new — créez un repo public nommé suppression-deta
3
Pousser le code
git remote add origin https://github.com/VOTRE_USER/suppression-deta.git
git branch -M main
git push -u origin main

ÉTAPE 2 — RENDER

1
Créer un compte Render
Allez sur render.com — inscription gratuite avec GitHub
2
Nouveau Web Service
Dashboard → New + → Web Service → Connectez votre repo GitHub
3
Configuration automatique
Le fichier render.yaml est détecté automatiquement. Render configure tout :
Build: npm install
Start: npm start
Health: /health
Port: 10000
4
Déployer !
Cliquez Create Web Service. Le build prend ~2 minutes.
Votre jeu sera live sur : https://suppression-deta.onrender.com
